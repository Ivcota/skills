## Phase 5 — The Four Thieves (defended)

### Thief 1 — Inability to Say No (pp. 167–173)

- [ ] **Three-Foot Rule** in place: request must connect to my ONE Thing — p. 170
- [ ] Default phrase drafted: **"No, for now."** — p. 181
- [ ] FAQ / script / checklist for 3 most-common requests — p. 170

### Thief 2 — Fear of Chaos (pp. 172–175)

- [ ] Partner / family briefed on what will be chaotic during blocks
- [ ] Reciprocal block-protection trade with one peer
- [ ] Willing to barter time to cover glass-ball domains

### Thief 3 — Poor Health Habits (pp. 175–179)

Daily Energy Plan — all five, in order (p. 177):
- [ ] 1. **Spiritual** — meditation or prayer
- [ ] 2. **Physical** — eat right, exercise, sleep
  - [ ] Nutritious breakfast
  - [ ] Weekly meals planned
  - [ ] **10,000 steps/day** tracked — p. 177
  - [ ] **8 hours of sleep** scheduled
- [ ] 3. **Emotional** — hug, kiss, laugh with loved ones
- [ ] 4. **Mental** — goals, plan, calendar
- [ ] 5. **Business** — ONE Thing time-block

### Thief 4 — Environment (pp. 177–182)

- [ ] Audited 5 closest people (who lifts, who drains?)
- [ ] Trial-walked the daily path to the bunker
- [ ] Sight thieves removed (papers, visible TV, visible phone)
- [ ] Sound thieves removed (notifications, open-door chatter)

- **Am I still on the Focusing Question's "can do" — not "should / could / would do"?** (p. 93)

If any answer is shaky, re-open the relevant reference file and re-run that section.
