# Lead Generation Checklist: Step-by-Step Worksheet

This is your working document for building a $100M lead generation engine from scratch. Work through each section in order. Fill in the prompts, score your system on each element, and use the templates to design your complete lead gen system. By the end of this checklist, you will have a clear, actionable plan for generating leads consistently and profitably.

## Lead Generation Scoring Rubric

Rate your lead generation system on each element. Be honest.

| Element | Score (1-10) | Notes / What to Improve |
|---------|-------------|------------------------|
| **Know Your Numbers** (LTGP, CAC, ratio calculated and tracked) | ___ | |
| **Core Four Execution** (at least one method, daily, consistently) | ___ | |
| **Lead Magnet** (solves narrow problem, bridges to core offer) | ___ | |
| **Rule of 100** (hitting 100/day consistently) | ___ | |
| **Tracking** (daily metrics, weekly reviews, data-driven decisions) | ___ | |
| **More, Better, New** (correct sequence, not chasing shiny objects) | ___ | |
| **Second Core Four** (or justified focus on one) | ___ | |
| **Customer Referrals** (systematically asking, tracking) | ___ | |
| **Lead Getters** (at least one active beyond referrals) | ___ | |
| **Compounding** (content growing, goodwill building, never stopping) | ___ | |
| **TOTAL** | ___ / 100 | |

### Score Interpretation

| Score Range | Assessment | Action |
|------------|-----------|--------|
| **90-100** | Lead generation machine. Multiple channels, compounding. | Optimize and scale. |
| **75-89** | Strong system. 1-2 elements need attention. | Improve lowest-scoring elements. |
| **60-74** | Good foundation. Multiple gaps. | Focus on the 3 weakest elements. |
| **40-59** | Below average. Significant work needed. | Go back to Steps 1-4. |
| **Below 40** | No system in place. | Start from Step 1. Commit to the Rule of 100. |

| 8 | Activate Lead Getters | Referral system live, first partner/affiliate engaged |
| 9 | Build the compounding machine | Content cadence set, goodwill strategy defined |
| 10 | Scale with Core Four on Steroids | Delegation plan, Lead Getters executing Core Four |
